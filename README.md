# My ML Utils

Collection of modular utility functions for data preprocessing, model evaluation, and pipeline automation.

## Installation
```bash
pip install git+[https://github.com/AbdelrhmanEbied/my-ml-utils.git](https://github.com/AbdelrhmanEbied/my-ml-utils.git)

## Project Structure
```text
my_ml_utils/
├── data/           # Data downloading and preprocessing (downloader, dataset_reporter)
├── engine/         # Training and evaluation logic (trainer)
├── metrics/        # Performance evaluation (accuracy, precision, recall, f1)
└── utils/          # Hardware setup, seeding, and checkpoint management
